# Альфа-реліз: IPTV Player 80s Edition

Перша альфа-версія сучасного IPTV-плеєра з неоновим Synthwave-дизайном!

## Що нового?
- Повністю оновлений графічний інтерфейс у стилі 80-х (Cyberpunk / Synthwave).
- Перегляд прямого ефіру (Live TV) та фільмів/серіалів (VOD).
- **Рекомендація:** Ця версія оптимізована та надійно працює з провайдерами **Xtream Codes**.

**Увага:** Програма знаходиться на стадії Альфа-тестування. Інтерфейс доступний лише українською мовою. Очікуйте багато нових оновлень найближчим часом!

Завантажте файл `IPTV-Player-Setup-1.0.0.exe` нижче для встановлення програми на Windows.
